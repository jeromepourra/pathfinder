## Requirements
- NodeJS

## Install packages
```bash
npm install
```

## Update packages
```bash
npm update
```

## Run project
- Build Javascript
```bash
npx tsc
```

- Build & watch change in Typescript
```bash
npx tsc -w
```

- Run PHP server
```bash
php -S localhost:8080
```

Next go to http://localhost:8080/index.php in your favorite browser
